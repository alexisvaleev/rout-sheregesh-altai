# Фото для маршрутов

Складывайте фотографии в эту папку.

## Правила именования

Файл: `{id-маршрута}-{номер}.jpg`

```
route-1-1.jpg      — маршрут "Две легенды Сибири", фото 1
route-1-2.jpg      — маршрут "Две легенды Сибири", фото 2
route-2-1.jpg      — маршрут "Первое знакомство с Сибирью", фото 1
route-3-1.jpg      — маршрут "Алтай за 3 дня", фото 1
route-4-1.jpg      — маршрут "❄️ Первое знакомство с Сибирью", фото 1
route-5-1.jpg      — маршрут "❄️ Алтай за 3 дня", фото 1
route-6-1.jpg      — маршрут "🎿 Grelka Fest", фото 1
```

## Формат
Рекомендуемый размер: 1200×800 px (соотношение 3:2)
Формат: JPG или PNG

## После добавления фото
Откройте `src/data/routePhotos.ts` и раскомментируйте / добавьте строки с `require()` для новых фото.
